뷰 기본용어(2)

Dec 18, 2019 in vue, vue 기본문법

1. 데이터 바인딩

데이터 바인딩이란 HTML 화면 요소를 뷰 인스턴스의 데이터와 연결하는 것을 말합니다. 데이터 바인딩 방법에는 v-bind, v-model등이 존재 합니다.

2. 디렉티브

디렉티브란 v- 접두사를 가지는 모든 속성들을 의미합니다.

디렉티브 종류

1. v-if
2. v-show 
3. v-for
4. v-bind
5. v-model
6. v-on
