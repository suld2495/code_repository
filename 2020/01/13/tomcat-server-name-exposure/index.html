톰캣 웹서버 정보 노출

Jan 13, 2020 in tomcat

Response 웹서버 정보 노출

서버로 요청을 보낸 후에 받은 Response에서 웹서버 정보가 노출이 됩니다.

노출 방지

웹 서버 정보 노출을 방지 하기 위해서는 server.xml의 Connector 설정을 통해 변경할수 있습니다. 태그에 server=””를 추가하고 값으로 노출을 원하는 문자열을 삽입하면 됩니다.
